# 🚀 Portfolio | Eduardo Reis — Full Stack Developer

Bem-vindo ao meu portfólio 👋  

Este projeto foi desenvolvido com o objetivo de apresentar, de forma clara e direta, meus projetos, minhas habilidades e minha evolução como desenvolvedor.

Mais do que um site, esse portfólio representa minha transição para a área de desenvolvimento e minha forma de pensar código: **organização, clareza e foco em solução.**

---

## 🧠 Sobre o Projeto

O portfólio foi pensado para ser simples, rápido e objetivo.

Aqui você encontra:

- Uma apresentação direta sobre quem eu sou  
- Meus principais projetos com foco prático  
- Tecnologias que venho utilizando no dia a dia  
- Um canal de contato  

Tudo isso com uma interface limpa e responsiva.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com foco em performance e boas práticas:

- **Frontend:** React.js, JavaScript (ES6+)  
- **Estilização:** CSS Modules (arquitetura organizada e reutilizável)  
- **Build Tool:** Vite  
- **Versionamento:** Git & GitHub  
- **Deploy:** Vercel  

---

## 💻 Funcionalidades

- 📱 Layout totalmente responsivo (Mobile First)  
- 🎯 Seção "Sobre Mim" com foco profissional  
- 🧩 Exibição de projetos de forma interativa  
- 📬 Formulário de contato  
- ⚡ Carregamento rápido e navegação fluida

## 🚀 Como rodar o projeto localmente

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/eduardo-olvreis/seu-repositorio.git
cd seu-repositorio
npm install
npm run dev
```

Depois, acesse no navegador:

http://localhost:5173
